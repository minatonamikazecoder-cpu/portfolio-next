# AI Assistant Skills Library

This project is configured with a curated set of `antigravity-awesome-skills` to help the AI assistant maintain the highest standard of code quality, performance, and UI/UX design.

These skills are stored locally in `../.agent/skills/` and are automatically loaded into the AI's context during development.

## Installed Skills

### Next.js & React Core
- **`nextjs-best-practices`**: Master-level guidance on App Router, server/client components, data fetching, and caching.
- **`nextjs-seo-indexing`**: Deep expertise in Next.js metadata API, sitemaps, robots.txt, and structured data generation.
- **`react-best-practices`**: Architecture patterns for scalable React applications.
- **`react-component-performance`**: Strategies for preventing unnecessary re-renders, optimizing hooks, and managing component lifecycles.

### Performance & Motion
- **`web-performance-optimization`**: Core Web Vitals mastery, bundle analysis, lazy loading, and rendering strategies.
- **`fixing-motion-performance`**: Specialized troubleshooting for Framer Motion, specifically regarding layout shifts, hydration errors, and GPU acceleration.

### Styling & Design
- **`tailwind-patterns`**: Scalable utility-class architectures, custom design systems, and responsive layouts.
- **`ui-ux-pro-max`**: High-end user interface design principles focusing on conversion, accessibility, and modern aesthetics.
- **`visual-emotion-engineer`**: Focuses on micro-interactions, color theory, and typographical rhythm to create "Awwwards-winning" emotional experiences.

## How to Use
When prompting the AI in future sessions, you don't need to do anything special. The AI will automatically read these skills and apply their principles to any code it writes for this portfolio.
